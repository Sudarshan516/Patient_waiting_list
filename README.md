# Patient_waiting_list_Analysis_Dashboard
# Project Introduction
This dashboard analyzes patient waitlist trends across multiple medical specialties from 2018–2021. It provides insights into patient demand, case types, and waiting time distribution using interactive visualizations.
# Project Goals
Track current status of pateint waiting list
Analyze historical monthly trend of waitinglist in inpatient & outpatient categories
Detailed specialty level & age profile analysis

# Key Project Insights
Bones, General, and ENT specialties have the highest patient waitlists.
Outpatient cases dominate the total waitlist, contributing the majority of patients.
Most patients fall in the 0–3 month and 3–6 month waiting period.
Outpatient demand increased steadily from 2018 to 2021, indicating rising healthcare service demand.
Older age groups (65+) show relatively higher wait times for some specialties.
 # Conclusion
The dashboard provides a comprehensive view of healthcare patient waitlists, helping stakeholders identify high-demand specialties and monitor waiting time trends. These insights can support resource planning, operational improvements, and better patient care management.
